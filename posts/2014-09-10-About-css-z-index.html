About css z-index

z-index生效的条件是首先要形成一个层叠上下文(stacking context)，条件如下：

  • 根元素(HTML)
  • 绝对（absolute）定位或相对（relative）定位且z-index值不为"auto",
  • 元素的opacity属性值小于1. (参考 the specification for opacity)

一个层叠上下文即一个上下层叠的单元，层叠的顺序如下：

  1. Background and borders of the root element 根层叠元素的背景和边框
  2. Descendant blocks in the normal flow, in order of appearance (in HTML) 子块元素，按照先后顺序，后者居上
  3. Floating blocks 浮动块
  4. Inline descendants in the normal flow 內联后代 e.g. 假如一个普通块元素有内容，它的兄弟浮动块层叠顺序居于它的上方，不过它的內联内容居于浮动元素的上方
  5. Descendant positioned elements, in order of appearance (in HTML) 定位元素，按照先后顺序，后者居上

同级别的层叠上下文按照z-index的大小，值较大者位于上。

同时一个层叠上下文可以想像成一个没有盖子的盒子，它所有的后代层叠上下文按照上述的级别由下到上排列，都在这个盒子里。和其他的层叠上下文就是不同盒子之间的层叠。即使一个本身顺序靠下的父层叠中有若干z-index的值较大的子层叠，这些子层叠依然是在父层叠形成的盒子里顺序层叠，它们依然不会飘到位置比它们父层叠高的其他层叠上方。
