css中height和line-height的区别

stackoverflow

  • height is the vertical measurement of the container. 一个容器的竖直方向的尺寸。
  • line-heightis the distance from the top of the first line of text to the top of the second. 多行文字中一行的上端到下一行上端的距离。 If used with only one line of text I'd expect them to produce similar results in most cases. 对于只有一行的情况是几乎没有区别。

    " I use height when I want to explicitly set the container size and line-height for typographic layout, where it might be relevant if the user resizes the text. "
