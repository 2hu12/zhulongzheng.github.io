About Bootstrap Grid System and Its Subtle Magic

I have been using bootstrap for a pretty long time, but just use it, don't understand it very deeply. At first I don't realize that I have to put a bunch of columns inside a row which is also should be inside a container. And I'm confused about the 15px padding of the container and -15px margin of the row, also the 15px padding of the column. But this is why the grid is subtle and excellent. Because it's perfect for nesting a row in a column.

THE SUBTLE MAGIC BEHIND WHY THE BOOTSTRAP 3 GRID WORKS

NESTING

When you have a container, row, and column set up, you can make new grids inside of a column. You add rows right inside the outer columns, you do not need to use them in containers: The trick here is that the column acts the same way as a container, it has 15px of padding that allows the row’s negative 15px of margin to overlay it… which in turn lets columns inside of this new row touch the edges, but push the content in with its own 15px padding, which also creates the gutter, as seen below.
