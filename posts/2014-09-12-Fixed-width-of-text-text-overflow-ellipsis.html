title: "Fixed width of text: text-overflow: ellipsis;" date: 2014-09-12 21:54:26 tags:

  • css
  • tips

css won't limit the num of characters, but it can limit the width and make ... at the end when the text is longer than the limited length.

.selector {
  text-overflow: ellipsis; /* will make [...] at the end */
  overflow: hidden; /* older browsers */
  width: 370px; /* change to your preferences */
  white-space: nowrap; /* paragraph to one line */
}

Attention: We need both of the text-overflow: ellipsis; and overflow: hidden;.

限制文字长度，超过结尾用三个点代表省略。 注意：text-overflow: ellipsis;和overflow: hidden;需要同时设置才有效果。

补充：固定table-cell的宽度设置 table.fixed { table-layout:fixed; }, 同时可以配合text-overflow实现省略效果。
